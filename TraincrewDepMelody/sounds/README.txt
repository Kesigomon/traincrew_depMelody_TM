# 音声ファイル配置ガイド

このディレクトリには、アプリケーションで使用する音声ファイル（MP3形式）を配置してください。

## ディレクトリ構造

sounds/
├── vehicle/        # 車両モード用の音声
│   ├── melody_up.mp3      # 上り列車メロディー
│   └── melody_down.mp3    # 下り列車メロディー
├── station/        # 駅モード用の音声
│   ├── sample_01.mp3      # サンプル駅1番線メロディー
│   ├── sample_02.mp3      # サンプル駅2番線メロディー
│   ├── test_01.mp3        # テスト駅1番線メロディー
│   └── test_02.mp3        # テスト駅2番線メロディー
└── door/           # ドア締まりますアナウンス
    ├── door_train.mp3     # 車両用ドア締まります
    ├── door_odd.mp3       # 駅用（奇数番線）
    └── door_even.mp3      # 駅用（偶数番線）

## 音声ファイルの要件

- フォーマット: MP3形式を推奨
- 音質: 適切なビットレート（128kbps以上推奨）
- 長さ: メロディーは5-30秒程度、アナウンスは3-10秒程度が適切

## プロファイルの編集

音声ファイルのマッピングは profiles/profile_default.csv で設定できます。
新しい駅や番線を追加する場合は、このCSVファイルを編集してください。

## 注意事項

音声ファイルが見つからない場合、アプリケーションはエラーをログに記録しますが、
クラッシュすることなく動作を継続します。
